## Button variants

### Primary

The style used for Call-to-action buttons, such as the 'Search' button or 'Get this media item' buttons.
It is a pink button.

### Action-menu

The styles used for header 'action-menu' buttons.

'action-menu' has no border and no background; on hover the background becomes dark-charcoal. It is used in the desktop header buttons.

'action-menu-inner' also has no border and no background. On hover, there is a light border. It is used inside the searchbar.

'action-menu-bordered' has a border but no background. On hover, the background becomes dark-charcoal. It is used in the desktop header buttons when the (old) header is scrolled.

'action-menu-muted' has a light charcoal background. It is used for filters when some filters are applied.
