# Icon

To display the icon, the path to the SVG must be specified. When using a bundler
like Webpack, importing an SVG (with the right loaders) will give the path to
that SVG.

```js
import iconSvg from '~/path/to/icon' // contains the URL path to the SVG
```

You can then pass this imported value as the `iconPath` prop.

```html
<VIcon :iconPath="iconSvg" />
```

<Canvas>
  <Story
    name="Default"
    args={{
      iconPath: replayIcon,
    }}
  >
    {Template.bind({})}
  </Story>
</Canvas>

Each icon SVG must have a `<g>` element right below the top-level `<svg>`
element. This element should have the `id` attribute equal to `"icon"`. If not,
pass the attribute value as the `gId` prop.

## RTL

`VIcon` can flip the icon for RTL languages. This is useful for certain icons
that have an intrinsic directional nature such as those involving arrows.

To see the RTL layout, switch to the Canvas tab and toggle the directionality
using the üåê button.
