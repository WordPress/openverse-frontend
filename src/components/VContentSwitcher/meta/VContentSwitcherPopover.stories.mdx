## v-model

The recommended way to use it is with `v-model` mapping to a `String`.

export const vModelTemplate = () => ({
  template: `
    <div>
      {{ item }}
      <VContentSwitcherPopover v-model="item"/>
    </div>
  `,
  components: { VContentSwitcherPopover },
  data() {
    return {
      item: 'all',
    }
  },
})
