## v-model

The recommended way to use it is with `v-model` mapping to a `String`.

export const vModelTemplate = () => ({
  template: `
    <div>
      {{ item }}
      <VSearchTypePopover v-model="item"/>
    </div>
  `,
  components: { VSearchTypePopover },
  setup() {
    const item = ref('all')
    return { item }
  },
})
